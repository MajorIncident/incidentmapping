# Roadmap

- [x] **Milestone 1 – Foundation**: Base canvas, ChainNode, persistence, tooling, smoke tests.
- [x] **Milestone 2 – Navigation & Shortcuts**: Sidebar inspector, keyboard-driven node creation/movement, undo/redo history.
- [ ] **Milestone 3 – Side Connections**: Side nodes and attachment edges, richer edge semantics (barrier annotations added).
- [ ] **Milestone 4 – Advanced Node Types**: Support for additional node kinds and annotations.
- [ ] **Milestone 5 – Reporting**: Export visuals, analytics summaries, integrations.
- [ ] **Milestone 6 – Enterprise Readiness**: Auth, RBAC, audit logging, scalability hardening.
